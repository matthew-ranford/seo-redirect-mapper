# SEO Redirect Mapper

A lightweight browser-based tool for managing SEO redirects during website migrations.

Upload a CSV with your existing slugs and new destination slugs — the tool identifies
what needs redirecting, suggests destinations using fuzzy matching for any blank rows,
and exports a clean CSV ready to upload to your CMS.

## Features

- CSV upload with auto column detection
- Fuzzy matching for blank destination slugs
- Editable redirect table with 301/302 toggle
- One-click export in Duda-compatible format

## Built with

- HTML, CSS, JavaScript (vanilla — no frameworks yet)
- Planned: React + Vite refactor
